運用ガイド（../index.html）に挿入する「Slack画面のスクショ」をこのフォルダに置いてください。
ファイル名は下記の通り（PNG推奨）。未配置の箇所はガイド上で点線のプレースホルダ表示になります。

detected.png      … 依頼メッセージに ❓（確認中）スタンプが付いた様子
confirm-dm.png    … botからの確認DM（🚩対応待ちにする / —不要 / 確認する ボタン）
due-modal.png     … 🚩を押すと開く期日設定モーダル（日付・時刻ピッカー）
flagged-card.png  … 「🚩 対応待ちに登録しました」カード（担当・タスク・期限）
reminder-dm.png   … リマインドDM（「本日が期日です」「🔴期日を過ぎています」等）
done.png          … 依頼メッセージに ✅（完了）スタンプを付けた様子
